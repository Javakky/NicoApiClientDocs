request(timeout: float = 400.0) → nicovideo_api_client.api.v2.result.SnapshotSearchAPIV2Result[ソース]

API にリクエストを送り、レスポンスを返す。

limit() で 100件を超える値が設定されている場合、複数回のリクエストに分けて実行される。 (そのため、 timeout の設定には注意が必要)

パラメータ

timeout -- リクエストを強制終了させるまでのタイムアウト時間。

複数回に分割リクエストが行われる場合でも、リクエストの合計時間とこの値を比較する。 また、timeout のうち 1/4 を接続時間に、 3/4 を読み込み時間に割り当てている。 :return: レスポンスオブジェクト
